# AuroraShare · 极光共享 - EXE 打包指南

## 📦 两种打包方式对比

| 特性 | build.bat（目录模式） | build_single.bat（单文件模式） |
|------|---------------------|-------------------------------|
| **输出文件** | 文件夹 + EXE | 单个 EXE 文件 |
| **文件大小** | ~30-40 MB | ~60-80 MB |
| **启动速度** | ⚡ 快（2-5 秒） | 🐌 较慢（10-30 秒） |
| **分发便利性** | 需要整个文件夹 | 单个文件，方便分享 |
| **推荐场景** | 正式部署、长期使用 | 临时演示、快速分享 |

---

## 🚀 快速开始

### 方法一：目录模式打包（推荐）

#### 步骤 1：运行打包脚本

双击运行：
```
build.bat
```

或在命令行执行：
```bash
cd d:\py\file-share
build.bat
```

#### 步骤 2：等待完成

脚本会自动完成以下操作：
1. ✅ 检查 Python 环境
2. ✅ 安装项目依赖
3. ✅ 安装 PyInstaller
4. ✅ 复制项目文件
5. ✅ 创建启动器
6. ✅ 打包为 EXE

#### 步骤 3：获取结果

打包完成后，EXE 文件位置：
```
dist_build\dist\AuroraShare-极光共享\AuroraShare-极光共享.exe
```

#### 步骤 4：测试运行

1. 进入上述目录
2. 双击 `AuroraShare-极光共享.exe`
3. 等待服务启动
4. 访问 http://localhost:8000

---

### 方法二：单文件模式打包

#### 步骤 1：运行打包脚本

双击运行：
```
build_single.bat
```

#### 步骤 2：等待完成

⚠️ **注意：** 单文件模式打包需要 3-5 分钟

#### 步骤 3：获取结果

打包完成后，文件位置：
```
release\AuroraShare.exe
```

#### 步骤 4：分发应用

可以将以下内容分发给用户：
- `release\AuroraShare.exe` （主程序）
- `release\使用说明.txt` （使用说明）

## 🔧 自定义配置

### 修改应用名称

编辑对应的 `.bat` 文件，找到：
```batch
--name "AuroraShare-极光共享"
```

修改为你想要的名称。

### 修改默认端口

编辑 `backend\config.json`：
```json
{
    "port": 8000
}
```

### 修改存储路径

编辑 `backend\config.json`：
```json
{
    "base_dir": "D:\\MySharedFiles"
}
```

### 添加应用图标

准备 `.ico` 图标文件，然后修改：
```batch
--icon=my_icon.ico
```

## ⚠️ 常见问题

### Q1: 打包时报错 "Module not found"

**解决方案：**
```batch
pip install fastapi uvicorn aiofiles python-multipart
```

然后重新运行打包脚本。

---

### Q2: EXE 启动后闪退

**可能原因：**
- 端口被占用
- 配置文件错误
- 权限问题

**解决方案：**
1. 检查 `config.json` 是否正确
2. 确保 8000 端口未被占用
3. 右键 EXE → 以管理员身份运行

---

### Q3: 浏览器无法自动打开

**解决方案：**
手动访问：http://localhost:8000

或检查系统默认浏览器设置。

---

### Q4: 上传的文件在哪里？

默认存储位置：
```
dist_build/dist/AuroraShare-极光共享/shared_files/
```

可以在 `config.json` 中修改 `base_dir` 参数。

---

### Q5: 如何让其他电脑访问？

#### 方案 A：局域网访问

1. 确保防火墙允许入站连接
2. 获取本机 IP：`ipconfig`
3. 其他电脑访问：`http://你的IP:8000`

#### 方案 B：公网访问

需要：
1. 路由器端口映射（8000 端口）
2. 或使用内网穿透工具（如 ngrok）

## 🎯 优化建议

### 减小 EXE 体积

1. **精简依赖**
   ```bash
   pip install --no-cache-dir -r requirements.txt
   ```

2. **删除不必要的模块**
   编辑 `.bat` 文件，移除不用的 `--hidden-import`

3. **使用 UPX 压缩**
   ```bash
   pip install pyinstaller[encryption]
   ```

### 加快启动速度

1. **使用目录模式**（比单文件快 5-10 倍）
2. **预加载常用模块**
3. **减少导入的库数量**

### 提升用户体验

1. **添加启动画面**
   ```python
   # 在 launcher.pyw 中添加
   import splash_screen
   ```

2. **系统托盘图标**
   ```python
   import pystray
   ```

3. **开机自启**
   创建快捷方式到启动文件夹

## 💡 高级技巧

### 创建安装包（Setup）

使用 Inno Setup 创建专业安装包：

```iss
[Setup]
AppName=AuroraShare
AppVersion=1.0
DefaultDirName={pf}\AuroraShare
OutputDir=installer

[Files]
Source: "dist_build\dist\AuroraShare-极光共享\*"; DestDir: "{app}"

[Icons]
Name: "{group}\AuroraShare"; Filename: "{app}\AuroraShare-极光共享.exe"
```

### 数字签名

使用证书对 EXE 进行签名：

```batch
signtool sign /f certificate.pfx /p password AuroraShare.exe
```

### 自动更新

实现自动更新功能：
1. 检测远程版本号
2. 下载更新包
3. 替换旧文件
4. 重启程序
