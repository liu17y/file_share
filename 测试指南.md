# 新功能快速测试指南

## 🎯 测试准备

1. 启动后端服务
```bash
cd backend
python main.py
```

2. 在浏览器中访问 `http://localhost:8000`

---

## ✅ 功能测试步骤

### 1️⃣ 测试重命名功能

**步骤：**
1. 上传或选择一个现有文件
2. 将鼠标悬停在文件上，右侧会出现操作按钮
3. 点击"编辑"图标（铅笔形状）
4. 在弹出的对话框中输入新名称
5. 按回车或点击"确定"

**预期结果：**
- ✓ 对话框正常弹出
- ✓ 当前名称已自动填充到输入框
- ✓ 重命名成功后文件列表自动刷新
- ✓ 显示"重命名成功"提示

**测试文件夹重命名：**
- 重复上述步骤，选择一个文件夹进行重命名

---

### 2️⃣ 测试拖拽上传功能

**步骤：**
1. 打开电脑的文件资源管理器
2. 选中一个或多个文件（建议先用小文件测试）
3. 拖拽文件到网页的文件列表区域
4. 松开鼠标

**预期结果：**
- ✓ 自动弹出上传对话框
- ✓ 显示"已添加 X 个文件，准备上传"提示
- ✓ 文件开始上传，显示进度条
- ✓ 上传完成后文件出现在列表中

**进阶测试：**
- 拖拽一个大文件（>50MB）测试分片上传
- 同时拖拽多个不同类型的文件
- 拖拽到空白区域 vs 拖拽到文件上

---

### 3️⃣ 测试返回上一级功能

**方法一：点击按钮**
1. 进入任意子文件夹
2. 点击左侧侧边栏的"返回上一级"按钮

**方法二：使用快捷键**
1. 确保不在输入框中输入
2. 按键盘的 `Backspace` 键

**预期结果：**
- ✓ 成功返回父目录
- ✓ 路径栏正确更新
- ✓ 文件列表显示父目录内容
- ✓ 在根目录时按钮为禁用状态
- ✓ 在根目录按 Backspace 提示"已在根目录"

**边界测试：**
- 在搜索模式下按 Backspace（应清除搜索）
- 在输入框聚焦时按 Backspace（应无效）

## 💡 使用技巧

### 重命名小技巧
- 支持批量重命名（选中多个文件后右键菜单）
- 重命名时可以使用中文、英文、数字和常见符号

### 拖拽上传技巧
- 可以从桌面直接拖拽
- 可以从其他文件夹窗口拖拽
- 支持一次拖拽多个文件

### 快捷键技巧
- `Backspace` - 快速返回上级（最常用！）
- 在搜索框输入时自动搜索，无需回车
