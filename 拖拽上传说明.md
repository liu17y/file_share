# AuroraShare · 极光共享 - 拖拽上传功能说明

## 🎯 功能概述

AuroraShare 现已支持**直观的拖拽上传**功能，您可以从桌面、文件管理器或其他应用直接拖拽文件到网页中，系统会自动开始上传。

---

## ✨ 核心特性

### 1️⃣ **明显的视觉提示**

当您将文件拖拽到文件区域上方时，会看到：

- 🔵 **蓝色虚线边框** - 整个文件区域高亮显示
- 💫 **半透明覆盖层** - 中央显示友好的提示图标和文字
- 📝 **清晰的文字说明** - "松开鼠标上传文件"
- 🎨 **美观的动画效果** - 平滑的弹出动画

### 2️⃣ **智能识别**

系统会自动区分：

- ✅ **外部文件拖拽** - 从桌面、其他应用拖入 → 自动触发上传
- ✅ **内部文件拖拽** - 在应用内拖拽文件 → 触发移动操作

### 3️⃣ **批量上传**

- 📦 一次可以拖拽多个文件
- 📁 支持拖拽整个文件夹（浏览器支持的情况下）
- ⚡ 自动分片并发上传

---

## 🖱️ 使用方法

### 方法一：从桌面拖拽

1. 打开文件资源管理器（Windows）或 Finder（Mac）
2. 选中一个或多个文件
3. 按住鼠标左键拖动
4. 将文件拖到 AuroraShare 的文件列表区域
5. 看到蓝色边框和提示后松开鼠标
6. 自动弹出上传对话框，显示进度

### 方法二：跨窗口拖拽

1. 并排打开浏览器和文件资源管理器
2. 从文件管理器中选中文件
3. 直接拖拽到浏览器窗口
4. 松开鼠标即可

### 方法三：多文件拖拽

1. 在文件资源管理器中框选多个文件
2. 或使用 Ctrl/Cmd 多选
3. 一次性拖拽到网页中
4. 所有文件自动加入上传队列

## 🔧 技术实现

### 前端事件监听

```javascript
// 拖拽经过文件区域
@dragover.prevent="onDragOverTable"

// 拖拽离开文件区域
@dragleave="onDragLeaveTable"

// 松开鼠标放置
@drop="onDrop"
```

### 状态管理

```javascript
data() {
    return {
        // 标记是否有外部文件正在拖拽到表格上方
        isDragOverTable: false
    }
}
```

### 智能判断逻辑

```javascript
onDragOverTable(event) {
    event.preventDefault();
    // 只有在非搜索模式且没有内部拖拽时才显示上传提示
    if (!this.dragSource && !this.searchMode) {
        this.isDragOverTable = true;
    }
}
```

### 外部文件处理

```javascript
onDrop(event) {
    event.preventDefault();
    
    // 检测是否有外部文件
    const files = event.dataTransfer.files;
    if (files && files.length > 0) {
        // 调用上传处理
        this.handleExternalFiles(files);
    }
}
```

## 🐛 常见问题排查

### 问题 1：拖拽没反应

**可能原因：**
- 浏览器不支持 HTML5 拖拽 API（IE 不支持）
- 当前处于搜索模式
- 正在进行内部文件拖拽

**解决方法：**
1. 使用 Chrome、Edge、Firefox 等现代浏览器
2. 退出搜索模式（清除搜索框）
3. 确保是从外部应用拖拽文件

### 问题 2：看不到拖拽提示

**可能原因：**
- CSS 样式未正确加载
- 暗色模式下对比度问题

**解决方法：**
1. 刷新页面（Ctrl+F5 强制刷新）
2. 检查浏览器控制台是否有 CSS 错误
3. 切换到亮色模式测试

### 问题 3：拖拽后文件没有上传

**可能原因：**
- `handleExternalFiles` 方法未定义
- 上传对话框组件有问题
- JavaScript 错误

**解决方法：**
1. 打开浏览器控制台（F12）查看错误
2. 确认 `index.html` 包含 `handleExternalFiles` 方法
3. 检查上传对话框组件是否正常加载

## 🔐 安全说明

### 文件类型限制

虽然支持拖拽任意文件，但系统可能有后端限制：

```python
# config.py 中可配置允许的类型
ALLOWED_EXTENSIONS = {'.txt', '.pdf', '.png', '.jpg'}
```

### 文件大小限制

- 单个文件最大：取决于服务器配置
- 建议：超过 1GB 的文件使用专用工具
- 临时存储：`.temp_uploads` 目录
