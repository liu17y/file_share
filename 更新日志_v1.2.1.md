##### 3. JavaScript 方法新增
```javascript
methods: {
    // 显示拖拽提示
    onDragOverTable(event) {
        event.preventDefault();
        if (!this.dragSource && !this.searchMode) {
            this.isDragOverTable = true;
        }
    },
    
    // 隐藏拖拽提示
    onDragLeaveTable(event) {
        const rect = event.target.getBoundingClientRect();
        const x = event.clientX;
        const y = event.clientY;
        
        if (x <= rect.left || x >= rect.right || 
            y <= rect.top || y >= rect.bottom) {
            this.isDragOverTable = false;
        }
    },
    
    // 处理外部文件上传（已有，增加提示关闭）
    async onDrop(event) {
        event.preventDefault();
        this.isDragOverTable = false; // 新增这行
        
        // ... 原有逻辑
    }
}
```

### 🔧 技术亮点

#### 1. 智能边界检测
```javascript
// 精确计算鼠标位置，只有真正离开区域才隐藏
const rect = event.target.getBoundingClientRect();
if (x <= rect.left || x >= rect.right || 
    y <= rect.top || y >= rect.bottom) {
    this.isDragOverTable = false;
}
```

#### 2. 状态冲突处理
```javascript
// 优先判断内部拖拽，避免误触发
if (!this.dragSource && !this.searchMode) {
    this.isDragOverTable = true;
}
```

#### 3. 性能优化
- 使用 CSS3 `transform` 而非 `top/left`
- `backdrop-filter` 实现毛玻璃效果
- 绝对定位避免重排

#### 4. 暗色模式支持
```css
.dark-mode .drag-overlay {
    background-color: rgba(31, 41, 55, 0.9);
}
```

### 🐛 已知问题

#### 问题 1：文件夹拖拽限制
某些浏览器可能不支持拖拽整个文件夹。

**解决方案：**
- 使用 Chrome/Edge 最新版本
- 或使用"上传文件夹"按钮

#### 问题 2：跨显示器延迟
在多显示器环境下可能有轻微延迟。

**解决方案：**
- 这是正常现象，不影响最终上传
- 确保显卡驱动最新
